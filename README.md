# SIEMENS team: Cybersecurity Knowledge Graph Code and Data

To contribute the code and data of the knowledge graph work:

- `git clone` this repo
- create a new branch
- add code and data (if data size is moderate). do not put large data files here.
- `git commit -m "added code and data by SIEMENS"`
- `git push -u origin <branch-name>`
- create a pull request on github website
